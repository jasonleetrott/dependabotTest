use this sample project to test any dependency package ignores. 

1. add your dependency to the project
2. add your Ignore to  buildTemplates/Dependabot/custom-repo-scans.yml
3. use the FB /feature/jlt/dependabotPackageIgnoreTest  or similar. 
4. manually run the  build [Dependabot-Single-repo-scan] with param
   dependencyFolder: /dependabot/packageUpdateTestSln/test
